# CAS-demo

## 使用C#简单实现单点登录功能

## C# + SQL + Redis

## 使用 重定向 和 HttpPost 实现单点登录单点登出。Redis暂存用户的登录信息.

## 客户端登录将会话SessionId发送给CAS-Server，登出时，根据SessionId进入客户端的私有会话清理用户登录状态.
